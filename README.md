# MEB Uyumlu Klasik Yazılı Oluşturucu

Bu proje, 5-8. sınıflar için ders, dönem, yazılı no ve kazanım seçerek klasik yazılı oluşturan basit bir web uygulamasıdır.

## Özellikler

- 5, 6, 7, 8. sınıf filtreleme
- Ders bazlı soru havuzu
- Dönem (1/2) ve yazılı no (1/2) seçimi
- Kazanım bazlı soru seçimi
- Yazılıyı yazdırma
- Yazılı metnini `.txt` olarak indirme

## Çalıştırma

Statik bir uygulamadır. Aşağıdaki komutla açabilirsiniz:

```bash
python3 -m http.server 4173
